# Project Brief

## Title
Smart Online Examination Request Scheduler

## One-line summary
A web application that simulates an online exam server and compares naive request handling with Round Robin CPU scheduling.

## OS concept
CPU Scheduling

## Real-world case
Online examination systems with many simultaneous student requests.

## Main deliverable
A browser-based demo that:
- generates student requests
- processes them in baseline mode
- processes them in Round Robin mode
- compares waiting time and turnaround time

- easy to explain

- can be implemented with simple tools
